NF-κB (Nuclear Factor kappa-light-chain-enhancer of activated B cells) är en transkriptionsfaktor som spelar en central roll i inflammation, immunförsvar och cellöverlevnad. Den aktiveras av olika signaler, såsom cytokiner (t.ex. NF-κB:s roll i immunsystemet

NF-κB är främst en del av det medfödda (innate) immunsystemet, men har även funktioner inom det adaptiva immunsystemet.

  1. Innate immune system ✅ (Huvudsaklig roll)

    • Aktiveras av Toll-like receptors (TLRs) vid infektion.

    • Styr transkriptionen av proinflammatoriska cytokiner (t.ex. TNF-α, IL-6, IL-1β).

  2. Adaptive immune system 🔶 (Sekundär roll)

Hur aktiveras NF-κB?

Den klassiska aktiveringsvägen involverar:

  1. Stimuli → Cytokiner (TNF-α, IL-1) eller patogener binder till receptorer (t.ex. TLRs).

  3. NF-κB frisläpps och translokerar till cellkärnan.

  4. Transkription av målgener → Cytokiner, adhesionsmolekyler och inflammatoriska mediatorer produceras.

Bra jobbat med att skapa ett GitHub-konto! Här är en steg-för-steg-guide för att ladda upp din hemsida till GitHub Pages:

Steg 1: Skapa ett nytt GitHub-repository

  1. Logga in på GitHub.

  2. Gå till Repositories (klicka på din profilbild uppe till höger och välj "Your repositories").

  3. Klicka på New för att skapa ett nytt repository.

  4. Fyll i följande:

    • Repository name: Välj ett namn, t.ex. twitch-streamers.

    • Description: (valfritt) En kort beskrivning av ditt projekt.

    • Välj Public.

    • Markera Initialize this repository with a README.

  5. Klicka på Create repository.

Steg 2: Ladda upp din HTML-fil

  1. Efter att ha skapat ditt repository, välj Add file och välj Upload files.

  2. Dra och släpp din HTML-fil (och eventuella andra filer, t.ex. CSS eller bilder) i uppladdningsområdet.

  3. Klicka på Commit changes när filerna har laddats upp.

Steg 3: Aktivera GitHub Pages

  1. Gå till fliken Settings för ditt repository.

  2. Scrolla ner till sektionen GitHub Pages.

  3. Under Source, välj main branch som källa.

  4. Klicka på Save.

Steg 4: Åtkomst till din hemsida

  1. När du har sparat inställningarna kommer GitHub Pages att generera en URL för din hemsida.

  2. URL:en kommer att vara: https://dittgithubanvändarnamn.github.io/repositorynamn/, t.ex. https://yourusername.github.io/twitch-streamers/.

Steg 5: Verifiera att din sida fungerar

  • Gå till URL:en som skapades för din hemsida och se om den visas korrekt.

Tips:

  • Om du uppdaterar sidan kan du göra en commit varje gång du ändrar något, t.ex. när du lägger till nya streamers.

  • Du kan också anpassa din URL genom att köpa en egen domän och koppla den till GitHub Pages om du vill.
